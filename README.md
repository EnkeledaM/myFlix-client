# myFlix-client

Client-side starter project for the myFlix app (CareerFoundry Achievement 3).
Built with **React** and **Parcel** to demonstrate the build process (bundling + transpilation).

## Project Structure

myFlix-client
- src/
  - index.html
  - index.jsx
  - index.scss

## Requirements

- Node.js (npm)

## Install

```bash
npm install



Run (Development)

Parcel is installed locally, so run it with npx:

npx parcel src/index.html


Then open:

http://localhost:1234

You should see "Good morning" rendered on the page.
