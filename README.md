# myFlix Client

Client-side React application for the myFlix app (CareerFoundry Achievement 3).

This application allows users to browse movies, view details, manage favorites, and update their profile. It connects to a custom REST API hosted on Heroku.

---

## Live Demo

https://relaxed-beignet-21ff1a.netlify.app

---

## Features

- User signup and login
- Browse a list of movies
- View detailed movie information
- Add and remove favorite movies
- Update user profile information
- Search and filter movies by title

---

## Tech Stack

- React
- React Router
- Parcel
- React Bootstrap
- Hosted on Netlify
- Backend API hosted on Heroku
