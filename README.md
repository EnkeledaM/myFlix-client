# myFlix Client

Client-side React application for the myFlix app (CareerFoundry Achievement 3).

## Features
- User signup and login
- Browse a list of movies
- Movie details view
- Add/remove favorite movies
- User profile update
- Search/filter movies by title

## Tech Stack
- React
- React Router
- Parcel
- React Bootstrap
- Hosted on Netlify
